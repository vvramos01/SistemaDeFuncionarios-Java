# Sistema de Funcionários

Projeto Java para demonstrar conceitos de programação orientada a objetos, incluindo herança, polimorfismo e encapsulamento. O sistema conta com uma hierarquia de classes onde Funcionario é a classe base e FuncTerceirizado é uma derivação com características específicas. Desenvolvido para fins educacionais, o projeto ilustra na prática como estruturar relações entre classes, calcular pagamentos de forma personalizada e organizar o código em pacotes.
